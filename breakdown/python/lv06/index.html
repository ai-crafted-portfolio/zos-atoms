    import 文の種類 （8 件）
    連番 技術項目 単一焦点要約 出典
    638 import モジュール名 モジュールを名前空間ごと読み込む。属性アクセスは mod.attr。 PYDOC
    639 from モジュール import 名前 特定属性を直接取り込む。複数同時 from m import a, b 可能。 PYDOC
    640 from モジュール import * ワイルドカードインポート。all で公開対象を限定可能。可読性のため通常推奨されない。 PYDOC

    643 相対インポート from .pkg import m 同パッケージ内の兄弟モジュールを参照。.. でさらに親へ。トップレベル実行スクリプトでは使えない。 PYDOC
