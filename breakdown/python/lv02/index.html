    167 elif による多分岐 elif は else if の短縮形。条件式を順次評価し最初に真となった節だけ実行する。 PYDOC
    168 条件式 (三項演算子) a if cond else b 代入や引数で値が必要な場面で 1 行 if-else。式として評価される。 PYDOC
    169 ネストした if と早期 return PEP 8 風には深いネストを避け guard clause (条件不一致なら return) を多用する。 PYDOC
    170 if での None 判定 is None None との比較は == ではなく is を使う。eq をオーバーライドした型で == None が想定外結果になる事故を避ける。 PYDOC
    171 if での bool 直接判定 if x == True は冗長で if x: が標準。逆に厳密な True/False 判定が必要なら is True を使う。 PYDOC
    172 if の評価順と短絡 and/or は左から評価し結果が確定した時点で評価を停止する。副作用のある関数呼び出しの順序に注意。 PYDOC
