  LISTDSD (List data set profile) Purpose: lists information from data set profiles. Short form: LD. Operands: DATASET('dataset-name'...) - specific profile names; PREFIX('prefix') - all profiles starting with prefix; ID(name) - profiles in name's access list; GENERIC/NOGENERIC; DSNS/NODSNS - list cataloged data sets covered; ALL - show full details; AUTHUSER - show access list; HISTORY/STATISTICS; NORACF (with segment names like DFP).
