    診断 (IPCS / Trace / ABEND)¶

    IPCS コマンド （38 件）
    連番 技術項目 単一焦点要約 出典
    4471 IPCS の位置付け Interactive Problem Control System。SVC ダンプ/SADMP/トランダンプを対話的に解析する z/OS 標準ツール。TSO セッション配下で起動 (IPCS コマンド)。 MF, AS2, ZOS31
    4472 SETDEF 現セッションで参照するダンプ・既定パラメータ (DSNAME, LENGTH, ASID 等) を設定。後続コマンドの省略時値となる。 AS2, ZOS31
    4473 SETDEF DSNAME SETDEF DSNAME('dump.dsn') で解析対象のダンプデータセットを既定指定。以降コマンドでは省略可能。 AS2, ZOS31
    4474 SETDEF PRINT/NOPRINT/LIST/NOLIST コマンド出力を IPCSPRNT データセットやログ DSN へ送るかの既定値。バッチ解析時は PRINT 推奨、対話時は NOLIST。 AS2, ZOS31
    4475 STATUS ダンプ取得時のシステム全体状況 (CPU/CSA/SQA/LPA/ASCB 概況・各種制御ブロック・PSW 等) を一括表示。最初の俯瞰に使う。 MF, AS2, ZOS31
    4476 STATUS REGISTERS 障害発生時の汎用レジスタ 0〜15 と制御レジスタを抽出表示。 AS2, ZOS31
    4477 STATUS FAILDATA ABEND を起こした単位 (タスク/SRB) の障害コンテキストを直接表示。 AS2, ZOS31
    4478 STATUS WORKSHEET 問題解析ワークシート用の主要属性 (Time, CPU model, IPL date, dump title 等) を集約。 AS2, ZOS31
    4479 SUMMARY アドレス空間ごとのタスク (TCB) と SRB 一覧、関連制御ブロックの要約を表示する基幹コマンド。 MF, AS2, ZOS31
    4480 SUMMARY FORMAT TCB ツリーを階層整形で表示。ASID/JOBNAME を絞り込むには ASID(x) を併用。 AS2, ZOS31
    4481 SUMMARY TCBERROR ABEND した TCB のみを抽出してエラー連鎖を表示。 AS2, ZOS31
    4482 VERBX Verb Exit。コンポーネント固有の整形ルーチンを呼び出す総合エントリ。VERBX サブネーム + オペランドで動作。 MF, AS2, ZOS31
    4483 VERBX MTRACE マスタートレース (コンソールメッセージ循環バッファ) を再生表示。ダンプ前後の WTO を時系列で参照。 AS2, ZOS31
    4484 VERBX LOGDATA LOGREC バッファに残っていたエラー記録を整形表示。ハードエラー/ソフトエラーを時系列確認。 AS2, ZOS31
    4485 VERBX SUMDUMP SVC ダンプ取得時に SDUMPX が収集した SUMDUMP 域 (主要制御ブロック圧縮コピー) を整形。 AS2, ZOS31
    4486 VERBX VSMDATA VSM (Virtual Storage Manager) の SQA/CSA/LSQA/SWA/サブプール統計を整形。仮想記憶枯渇調査の起点。 AS2, ZOS31
    4487 VERBX GRSTRACE GRS (Global Resource Serialization) の ENQ/DEQ ホルダ/ウェイタ チェーンを整形。ENQ デッドロック解析。 AS2, ZOS31
    4488 VERBX RSMDATA RSM (Real Storage Manager) のフレーム使用状況、ページング統計を整形。 AS2, ZOS31
    4489 VERBX TCBERROR ABEND チェーン (RTM2WA) を分かりやすく整形。PSW/レジスタ/完了コードを表示。 AS2, ZOS31
    4490 VERBX BPXMTRCE USS (z/OS UNIX) カーネルトレースを整形表示。OMVS 障害解析の基本。 AS2, ZOS31
    4491 LIST 指定アドレスから記憶域をダンプ表示。LIST addr LENGTH(n) ASID(x) で長さと AS を指定。 MF, AS2, ZOS31
    4492 LIST 構造化表示 LIST addr STRUCTURE(name) で制御ブロック名 (ASCB,TCB,RB,CDE 等) を指定すると DSECT に従い整形表示。 AS2, ZOS31
    4493 FIND 現在表示中のダンプ域内を文字列/16 進パターンで検索。FIND 'JOBNAME' で次出現へジャンプ。 AS2, ZOS31
    4494 WHERE WHERE addr で指定アドレスが属するモジュール/CSECT/オフセットを特定。PSW やレジスタ値の解釈に必須。 MF, AS2, ZOS31
    4495 NAME シンボリック名 (CVTPTR, ASCBJBNI 等) と実際のオフセット/絶対アドレスとを相互変換。 AS2, ZOS31
    4496 EVALUATE EVALUATE addr で算術式やシンボルを評価し、結果アドレスを表示。間接参照 (%addr) 含む。 AS2, ZOS31
    4497 COPYDUMP SYS1.DUMPxx 配下の自動取得ダンプを別 DSN へコピーして長期保管・転送用に切り出す。 AS2, ZOS31
    4498 SCAN / DROPDUMP SCAN はダンプ内主要制御ブロック (CVT/ASCB/TCB/RB 等) の整合性チェックで破損検出、DROPDUMP は Dump Inventory から登録解除 (物理 DSN 残存モードあり)。 AS2, ZOS31
    4499 EQUATE セッションローカルなシンボル (例: EQUATE MYTCB X'00C8F300') を定義。後続コマンドで使い回し。 AS2, ZOS31
    4500 NOTE / DELETE NOTE で解析メモ記録、DELETE で EQUATE/NOTE 削除。セッション再開時の注釈管理。 AS2, ZOS31
    4501 OPEN / CLOSE OPEN DSNAME('dump') で IPCS セッションに開き、CLOSE で閉じる。複数ダンプ切替解析で必須の対。 AS2, ZOS31
    4502 SELECT SELECT ASID(x) で表示対象アドレス空間を絞り込み、後続 SUMMARY/LIST に影響。 AS2, ZOS31
    4503 LISTDUMP Dump Inventory に登録されている全ダンプ DSN を一覧表示。 AS2, ZOS31
    4504 SYSTRACE システム・トレース (常時動作のハードウェア&ソフトウェアイベント循環バッファ) を整形再生する IPCS コマンド。 MF, AS2, ZOS31
    4505 SYSTRACE ALL SYSTRACE ALL で全 CPU 全 ASID のトレースを時系列マージ表示。 AS2, ZOS31
    4506 SYSTRACE ASID/TIME SYSTRACE ASID(x) で特定 AS のみ、TIME(LOCAL/GMT) で時刻表現を切替。 AS2, ZOS31

    4508 RUNCHAIN ポインタチェーンを自動追跡しながら順次 LIST。TCB → RB → … 等の連鎖解析。 AS2, ZOS31

    4518 SDUMPX マクロ 現代版 SVC ダンプ取得マクロ。SUMDUMP 採取、SDATA オペランドでデータ種別 (CSA,SQA,LPA,GRSQ,RGN,TRT) を選択。 AS2, ZOS31
    4519 SDATA オペランド ダンプ含めるデータ範囲を指定 (CSA, SQA, LSQA, LPA, NUC, PSA, SUM, TRT, ALLNUC, ALLPSA, GRSQ, RGN, WLM, XESDATA 等)。 AS2, ZOS31
    4520 SDUMP の出力先 SYS1.DUMPxx (xx=00-99) に自動シリアル割当。割当不可なら『SVC DUMP NOT TAKEN』。 AS2, ZOS31
    4521 DUMPDS コマンド DUMPDS NAME=,ADD/CLEAR/DEL でダンプデータセットの追加・解放・名前パターン変更をオペレータが実行。 AS2, ZOS31
    4522 CHNGDUMP コマンド CHNGDUMP SET,SDUMP=… で SDUMP の既定 SDATA、サイズ上限、抑止対象 ASID を動的変更。 AS2, ZOS31
    4523 DUMP コマンド (オペレータ) DUMP COMM='title',JOBNAME=,ASID=,SDATA=… でオペレータが手動 SVC ダンプ要求。 AS2, ZOS31

    4536 SYSUDUMP DD 通常用途向けの小容量タスクダンプ。問題ジョブのアドレス空間配下の限定領域を整形出力。 MF, BAS, AS2, ZOS31

    4539 ABDUMP 抑止 JCL に該当 DD を書かないか DUMMY 指定で ABDUMP を抑止可能。ただし障害解析のためには通常残す。 BAS, AS2, ZOS31

    4541 IEADMP00 (SYSUDUMP) SYS1.PARMLIB(IEADMP00) で SYSUDUMP の既定領域を制御。 AS2, ZOS31
    4542 IEADMR00 (SYSMDUMP) SYS1.PARMLIB(IEADMR00) で SYSMDUMP の SDATA 既定を制御。 AS2, ZOS31
