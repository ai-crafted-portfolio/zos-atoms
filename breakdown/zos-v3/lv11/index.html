    IEFBR14 （5 件）
    連番 技術項目 単一焦点要約 出典
    2686 プログラム本体の役割 BR 14 (Branch Register 14, 復帰) 1 命令だけを実行して RC=0 で終了する最小のロードモジュール。プログラム自体は何もしないため、DD 文の割当て・解放だけが処理対象となる。 BAS, AS1, ZOS31
    2687 DISP=(NEW,CATLG) でのデータセット作成用途 PGM=IEFBR14 と DD 文 (DISP=(NEW,CATLG,DELETE)) を組み合わせてデータセットを「割り当てるだけ」のジョブステップを書く典型用途。 BAS, AS1, ZOS31
    2688 DISP=(OLD,DELETE) でのデータセット削除用途 既存データセットを DD で指定し、ステップ終了時に DELETE してカタログから抹消する用途。SCRATCH/UNCATLG ユーティリティ無しでもデータセット削除が可能。 BAS, AS1, ZOS31
    2689 戻りコード (RC) は常に 0 BR 14 だけを実行するため、IEFBR14 自身が異常 RC を返すことはない。ただし DD 割当て失敗 (JCL ERROR) や DISP 処理失敗時にはステップ自体が異常終了する。 BAS, ZOS31
    2690 PARM 指定は無視される PARM= を書いても IEFBR14 は何も処理しないので無効。ダミーステップでパラメータを明示したい場合のみ書く慣習はある。 BAS, MF

    2691 ユーティリティの基本機能 順次データセット (PS) の単純コピー・印刷・PDS メンバー化を行う標準ユーティリティ。SYSUT1 から SYSUT2 へレコードを転送する。 BAS, AS1, ZOS31
    2692 SYSUT1 DD (入力) コピー元の入力データセットを指定する DD。順次データセット、PDS メンバー、SYSIN 形式の埋め込みデータも入力にできる。 BAS, AS1, ZOS31
    2693 SYSUT2 DD (出力) コピー先の出力データセットを指定する DD。順次データセット、SYSOUT、PDS メンバーが指定可能。 BAS, AS1, ZOS31
    2694 SYSPRINT DD (メッセージ出力) ユーティリティのメッセージと統計情報を出力する DD。SYSOUT=* が典型値。省略すると JCL ERROR。 BAS, AS1, ZOS31
    2695 SYSIN DD (制御文) GENERATE / RECORD / MEMBER などの制御文を入力する DD。単純コピーであれば DD DUMMY または //SYSIN DD DUMMY とする。 BAS, AS1, ZOS31
    2696 SYSIN DD DUMMY (制御文無しの単純コピー) 制御文を全く与えない場合は SYSIN DD DUMMY を書く。SYSUT1→SYSUT2 のレコード単純コピーが実行される。 BAS, AS1, ZOS31
    2697 GENERATE 制御文 RECORD/MEMBER で部分編集を行う場合に冒頭に置く宣言文。MAXFLDS=n / MAXNAME=n で編集領域と名前テーブルサイズを宣言する。 BAS, MF, ZOS31
    2698 GENERATE MAXFLDS オペランド 後続 RECORD 制御文の FIELD パラメータ数の合計を申告する。実際の合計以上の値を指定する必要があり、不足すると編集が打ち切られる。 BAS, MF, ZOS31
    2699 GENERATE MAXNAME オペランド MEMBER 制御文で指定するメンバー名の総数を申告する。PDS の複数メンバーを一度に作成する際に必須。 BAS, MF, ZOS31
    2700 GENERATE MAXLITS オペランド FIELD パラメータで使用するリテラル (定数文字列) の総バイト数を申告する。リテラル挿入を行う編集で必要。 BAS, MF
    2701 MEMBER 制御文 出力先 PDS の中で新しく作成 (または置換) するメンバー名を指定する。続く RECORD 制御文の対象メンバーとなる。 BAS, MF, ZOS31
    2702 MEMBER NAME=メンバー名 MEMBER 制御文のオペランド。8 文字以内のメンバー名を指定する。複数 MEMBER を並べて 1 度の実行で複数メンバーを生成できる。 BAS, MF, ZOS31
    2703 RECORD 制御文 コピー対象となる入力レコード範囲と編集内容を指定する。IDENT / FIELD オペランドで「どこから」と「何を編集」を表現する。 BAS, MF, ZOS31
    2704 RECORD IDENT オペランド (長さ,'文字列',位置) の 3 要素で識別レコードを指定する。指定した識別レコードに到達するまで前段の編集を続け、到達後に次の RECORD ブロックへ移る。 BAS, MF, ZOS31
    2705 RECORD FIELD オペランド (長さ,入力位置,変換,出力位置) の 4 要素で 1 フィールドの編集を指定する。1 個の RECORD 制御文に複数 FIELD を並べられる。 BAS, MF, ZOS31
    2706 RECORD FIELD 変換コード PZ (Packed→Zoned)、ZP (Zoned→Packed)、HE (16進→EBCDIC) などの変換コードを指定する。省略時は変換無しのバイト単位コピー。 BAS, MF
    2707 RECORD FIELD リテラル挿入 入力位置の代わりに 'C''abc''' のようなリテラル指定で固定文字列を出力に埋め込むことができる。GENERATE MAXLITS で総量を申告する。 BAS, MF
    2708 DCB 不一致時の動作 SYSUT1 と SYSUT2 で RECFM/LRECL が異なる場合、可能な範囲で IEBGENER が変換する (例: 固定長→可変長)。互換性が無い組合せでは異常終了する。 BAS, ZOS31
    2709 PDS メンバーへの出力 (DSN=lib(mem)) SYSUT2 で DSN=LIB(MEM) を指定すれば 1 メンバーを単純コピーで作成できる。GENERATE/MEMBER 不要。 BAS, AS1, ZOS31
    2710 SYSOUT への印刷用途 SYSUT2 を SYSOUT=A や SYSOUT=* にすれば順次データセットをそのまま印刷キューに送れる。SDSF で見るデバッグ出力用途で頻出。 BAS, AS1, ZOS31
    2711 ICEGENER (DFSORT 高速版) との関係 DFSORT 導入環境では IEBGENER の SVC が ICEGENER に置換され、内部で SORT COPY 相当に高速実行されることがある。JCL 上は IEBGENER のままで透過置換される。 MF, ZOS31
    IEBCOPY （22 件）
    連番 技術項目 単一焦点要約 出典
    2712 ユーティリティの基本機能 PDS / PDSE の全体コピー、メンバー選択コピー、圧縮、UNLOAD/LOAD を行う標準ユーティリティ。SYSPRINT / SYSUT1〜SYSUT4 / 複数の INDDn・OUTDDn を使い分ける。 BAS, AS1, ZOS31
    2713 SYSPRINT DD ユーティリティのメッセージ・統計情報出力先。省略不可。SYSOUT=* が典型値。 BAS, AS1, ZOS31
    2714 SYSIN DD COPY / SELECT / EXCLUDE / COMPRESS などの制御文の入力 DD。 BAS, AS1, ZOS31
    2715 SYSUT3 / SYSUT4 ワーク DD IEBCOPY が内部で使うワークスペース DD。PDS 大規模コピーや圧縮で必要となるが、PDSE 専用処理では不要なケースもある。 BAS, MF
    2716 COPY 制御文 コピー処理の宣言。INDD と OUTDD を指定して入力 PDS と出力 PDS を結合する。1 つの COPY ブロックに複数 INDD/OUTDD を並べられる。 BAS, AS1, ZOS31
    2717 COPY INDD オペランド INDD=ddname または INDD=((ddname,R)) 形式で入力 PDS DD 名を指定する。R オプションは REPLACE (同名メンバーの上書きを許可)。 BAS, AS1, ZOS31
    2718 COPY OUTDD オペランド OUTDD=ddname で出力 PDS DD 名を指定する。1 つの COPY ブロックは 1 つの OUTDD を持つ。 BAS, AS1, ZOS31
    2719 COPY INDD の R オプション (REPLACE) 出力側に同名メンバーが既存でも置換する。R オプション無しの場合は既存メンバーをスキップ (NOT SELECTED) する。 BAS, AS1, ZOS31
    2720 SELECT 制御文 コピー対象を「指定メンバーのみ」に絞り込む。MEMBER=(m1,m2,...) でメンバー名リストを与える。 BAS, AS1, ZOS31
    2721 SELECT MEMBER の改名形式 ((old,new,R)) (元名,新名,R) のタプルで指定すると、コピー時にメンバー名を改名できる。R を付けると出力側の同名メンバーを置換。 BAS, AS1, ZOS31
    2722 SELECT の R 個別指定 (memname,,R) と中間を空欄にすれば改名せず置換指定だけが可能。INDD の R オプションをメンバー単位で上書きする。 BAS, MF
    2723 EXCLUDE 制御文 「指定メンバー以外」をコピーする否定指定。MEMBER=(m1,m2,...) で除外メンバー名を列挙する。SELECT と同時には使えない。 BAS, AS1, ZOS31
    2724 COPYGRP 制御文 (プログラムオブジェクト用) PDSE のプログラムオブジェクト (program object) を別名・エイリアスを含めて 1 グループとしてコピーする。LOAD モジュールの正しいコピーに必要。 BAS, MF, ZOS31
    2725 COPYGRP と COPY の違い COPY はエイリアスを単独メンバー扱いするのに対し、COPYGRP は本体とエイリアス・拡張属性を 1 セットでコピーする。プログラム実体の整合性を保つために PDSE では COPYGRP を推奨。 MF, ZOS31
    2726 COMPRESS 制御文 PDS の死領域 (削除済みメンバーの未回収領域) を回収する。OUTDD=ddname で対象 PDS を指定する。PDSE には不要 (自動管理)。 BAS, AS1, ZOS31
    2727 COMPRESS は in-place 実行 対象 PDS 上で直接圧縮するため、途中で障害が起きるとディレクトリ破損のリスクがある。重要 PDS は事前バックアップ推奨。 BAS, ZOS31
    2728 UNLOAD (アンロード) 機能 PDS / PDSE を順次形式のバックアップデータセットに書き出す。出力 DD が PS (順次) ならば自動的に UNLOAD 形式となる。 BAS, AS1, ZOS31
    2729 LOAD (リロード) 機能 UNLOAD で生成された順次形式バックアップから PDS / PDSE を復元する。入力 DD が PS、出力 DD が PDS/PDSE の場合に自動判定。 BAS, AS1, ZOS31
    2730 PDS → PDSE 変換 INDD が PDS、OUTDD が PDSE であれば、IEBCOPY 経由でフォーマット変換コピーが行われる。逆方向も可。 MF, ZOS31
    2731 PRINT 制御文 (メンバー一覧出力) MEMBER オペランドで指定したメンバーをダンプ形式で SYSPRINT に出力する。デバッグ用途。 MF
    2732 REPLACE オプション (=R) COPY/SELECT 双方で使える「既存メンバーを置換する」フラグ。R 無しでは既存メンバーはコピーをスキップされる。 BAS, AS1, ZOS31
    2733 ALIAS 自動コピーの挙動 通常の COPY ではエイリアスの存在は認識するが、本体メンバーがコピー対象に含まれていないと一貫性が壊れる。COPYGRP を使うか、SELECT に本体とエイリアスの両方を明示する必要がある。 MF, ZOS31
