    デバイス管理 / lsdev （12 件）
    連番 技術項目 単一焦点要約 出典
    1201 lsdev (引数なし) ODM の CuDv (Customized Devices) を読み、システムが認識している全デバイスを location/status とともに一覧表示 AIX1, AIX73
    1202 lsdev -C Customized デバイス (実際にシステムに構成済み) のみを一覧表示。-C は明示指定でも既定でも動作する基本オプション AIX1, AIX73
    1203 lsdev -P Predefined (PdDv) デバイスを表示。サポートされているデバイスクラス/タイプの定義一覧を確認するときに使用 AIX1, AIX73
    1204 lsdev -C -c class クラス指定で絞込 (例: disk, adapter, tape, cdrom, tty, lan, pty, if, sys, memory) AIX1, AIX73
    1205 lsdev -C -t type タイプ指定で絞込 (例: type=osdisk, scsd, fcp, ent, en) AIX1, AIX73
    1206 lsdev -C -s subclass サブクラス指定で絞込 (例: scsi, fcp, sas, pci, vscsi, vdevice) AIX1, AIX73
    1207 lsdev -C -l name 特定デバイス名 (hdisk0, en0, fcs0 等) のみ表示 AIX1, AIX73
    1208 lsdev -C -p parent 親デバイス名で絞込 (例: -p scsi0 → scsi0 配下のデバイス) AIX1, AIX73
    1209 lsdev -H ヘッダ行 (name / status / location / description) を付けて表示 AIX73
    1210 lsdev -F format 出力フィールドを書式指定 (例: -F 'name status location')。スクリプト連携時に多用 AIX1, AIX73
    1211 lsdev -r class / -r subclass / -r type PdDv に登録された有効な class/subclass/type の値を列挙 AIX73
    1212 デバイス state (Available / Defined) Available=構成済み、Defined=ODM 定義のみで未構成。lsdev 出力 2 列目で確認 AIX1, AIX73
