    連番 技術項目 単一焦点要約 出典
    2109 ksh -l (ログインシェル) -l オプションを付けると ksh はログインシェルとして起動し、/etc/profile および ~/.profile を読み込む。AIX で su - user した直後の挙動と同等。 KORN, AIX73
    2110 ksh -i (対話シェル) -i オプションは対話モードを強制する。スクリプト経由でも端末入力を受け付ける状態にし、PS1 の表示や job control を有効化したい場面で利用する。 KORN
    2111 ksh -r (制限シェル) -r は restricted ksh (rksh) として起動する。cd / PATH 変更 / リダイレクト >, >> 等が禁止され、kiosk 端末や受託オペ用に最低限のコマンド実行のみを許可する。 KORN, AIX73
    2112 ksh -s (標準入力読込) -s は引数で渡されたファイルではなく標準入力からスクリプトを読む。位置パラメータは -s 以降の引数に割り当てられる。 KORN
    2113 ksh -c '文字列' -c は引数で渡された文字列を 1 つのスクリプトとして実行する。シェルワンライナーや system() 呼び出しのような利用方法。 KORN
    2114 ksh -x (xtrace) -x は PS4 をプレフィックスにして実行コマンドを stderr に出す。デバッグ目的で多用される。 KORN, AIX73
    2115 ksh -v (verbose) -v は読み込んだ各行を stderr にエコーする。スクリプトのソース表示が目的で xtrace とは粒度が異なる。 KORN
    2116 ksh -e (errexit) -e は コマンドが非ゼロを返した瞬間にシェルを終了させる。スクリプトの fail-fast 化に使う。 KORN, AIX73
    2117 ksh -u (nounset) -u は未定義変数の参照をエラーにする。typo を早期検出する。 KORN
    2118 ksh -n (noexec) -n はコマンドを実行せず構文だけチェックする。デプロイ前の構文検証に使う。 KORN
    2119 ksh -f (noglob) -f はパス名展開 (ワイルドカード) を抑制する。* や ? をリテラルとして渡したい場面に。 KORN
    2120 ksh -a (allexport) -a は以降に代入される変数を自動的に export する。環境引き渡しを一括で行いたい時に使う。 KORN
    2121 ksh -h (trackall) -h は エイリアス使用時にパスをハッシュ化して以後の検索を省く。 KORN
    2122 ksh -m (monitor) -m は job control を有効化する。対話シェルでは標準有効だが、非対話シェルでもジョブ管理したい時に明示する。 KORN
    2123 ksh -A 配列名 値リスト -A は スクリプト引数を配列に格納して起動する。 KORN
    2124 ksh -o オプション名 -o は set -o と同じ長い名前のオプションを起動時に指定する。例: ksh -o vi で起動直後から vi モード。 KORN
    2125 ksh --version ksh93 のビルド情報を表示する。AIX73 同梱は ksh93 (例: Version JM 93u+ など)。 AIX73
    2126 /bin/ksh と /usr/bin/ksh AIX73 では /usr/bin/ksh が ksh93。/bin/ksh は同じバイナリへのシンボリックリンク。シェバン #!/usr/bin/ksh を推奨。 AIX73
    2127 ksh88 vs ksh93 AIX73 同梱は ksh93。ksh88 互換は ksh ではなく bsh 寄り。配列・浮動小数・連想配列・compound variable は ksh93 のみ。 KORN, AIX73
    2128 /etc/environment AIX 固有。ログイン時に PATH や TZ を一括設定する場所。/etc/profile より早く読まれる。 AIX73
