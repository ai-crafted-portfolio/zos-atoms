# RMF (Resource Measurement Facility)

**製品**: z/OS System Programming ／ **本ページ件数**: 95 件 ／ **出典区分**: 初版（事前知識補完を含む）

!!! warning "初版コンテンツ"
    本ページは初版です。一部の技術項目は一次資料（公式マニュアル・Redbook 等）
    との突合を予定しており、出典欄に「一次資料 突合予定」と表示される項目があります。
    内容は実機運用前に必ず最新の一次資料で確認してください。
