### 検証手順 ・ 理解度チェック

??? example "#7387 TCP/IP Considerations"

    **検証手順**

    検証目的: NetView 6.4 における 'TCP/IP Considerations' の意味・配置・操作方法を関連マニュアル記述に従って実機確認する。

    ```text
    [セッション環境]
    NetView 6.4 オペレータコンソール (NCCF)

    [コマンド] NetView コンソール: TCPCONN START (or LIST STATUS=DVIPA)
    [出力] TCP/IP 接続データ収集開始メッセージ

    [マニュアル抜粋] (出典: NetView_6.4_Installation_Migration_Guide.pdf p.69 原文ママ)
      DSITCPCF: defines the initialization values for the DSITCPIP task. Do not modify the Data REXX version. DSITCPRF: defines TCP/IP operator security profiles. The WEB_SERVER statement that defined the encryption keys for HTTP server sessions is removed.
    ```
